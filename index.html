Question-1: What is the difference between localStorage and sessionStorage?

Answer: Difference between localStorage and sessionStorage are given below:

Local Storage Session Storage
Local Storage is a kind of storage which is given by the browser to use store some data for further uses. Session storage is also a kind of storage which is given by the browser to use for a particular session.
Local storage data is stored for a certain time until the user manually delete it or the browser delete automatically after a certain time. Session storage is stored for a particular session until the tab or browser closed.
Local storage data can be read after reopen of the browser. Session storage can not be read after reopen of browser.
Question-2: How does Javascript work?

Answer: Javascript is a dynamic scripting language. Now a days, JS is considered as the heart of the modern web. Javascript runs it code as a synchronus way, that is in a specific order of execution and Javascript is also known as single threaded language.

Javascript codes are executed in Execution Context which is like as a container where all the codes are executed. In Execution Context, all kinds of variables and functions are stored in a (key,value) pair in Memory and Code components executed one line at a time.
